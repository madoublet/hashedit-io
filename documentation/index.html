Installation

Hashedit provides you with a starting point for creating applications running on the Express framework using Node JS. This starting point is called Hasedit Spark. The basic process for installing Hashedit is shown below.

> git clone https://github.com/madoublet/hashedit-spark .
> cp config.sample.js config.js
> npm install

Updating config.js

The config.js which you copied from config.sample.js during the installation contains settings to run and authenticate your application.

var exports = module.exports = {};

// app configurations
exports.app = {
    "url": "https://my-hashedit-site.io",
    "port": 3000
};

// setup google authentication
exports.google = {
    "clientId": "GOOGLE_CLIENT_ID",
    "clientSecret": "GOOGLE_CLIENT_SECRET"
};

// setup authorized users
exports.authorized = [
{
    "email": "sample@gmail.com",
    "provider": "google"
}
];
Setting Description
app.url The URL for your app. This is used as a base for the return URL from Google during authentication.
app.port The port your app will run on.
google.clientId The Client ID provided by Google when setting up authentication. See Using OAuth2 for Web Server Applications on how to setup the client id and secret.
google.clientSecret The Client Secret provided by Google when setting up authentication.
authorized An array of authorized emails that can update pages. Add your gmail account here to authorize it. The only provider supported right now is Google. Others will be added in the future.
