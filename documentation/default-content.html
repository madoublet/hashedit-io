Theming Sites

Hashedit makes it easy to maintain a consistent look-and-feel across your site.

Default Content

The primary mechanism for maintaining a consistent look-and-feel is default content. By specifying default content, you ensure that new pages created in Hashedit match the look-and-feel of your site. You can set default content by updating the .default.html (note the leading dot) file in your root directory (e.g. public/.default.html).

You can also specify a default content file in each subdirectory of your site. This will ensure that all files in that directory are created using that default content (e.g. public/page/.default.html or public/product/.default.html).
