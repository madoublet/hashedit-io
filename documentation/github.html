Github Repositories

Hashedit was built from the ground-up to be modular. So, you can use the whole stack, just the editor, or the static website API.

Hashedit Spark

The goal of the Hashedit Spark repository is to provide you with a full stack starting point for your Hashedit website. It contains the Express JS parts and brings in the other two Hashedit repos as dependencies.

madoublet/hashedit-spark

Static Kit

The goal of the Static Kit repository is to provide you with a REST API for authenticating a user via a Google login and access to static resources four your website. It is specifically built around the needs of Hashedit but is generic enough that it could be used independently of the editor.

madoublet/static-kit

Hashedit

The Hashedit repository is the client library that implements the visual editor. It can be used in conjunction with Hashedit Spark, but you can also modify it for your own needs.

madoublet/hashedit
